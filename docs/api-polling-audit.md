## Polling intentionally retained

CPU, memory and network bandwidth remain live dashboard metrics. Pool hooks again default to 30s polling for live pool health, with React Query observers stopping automatically when their components unmount and background-tab polling disabled. Integrated Storage pool-detail polling remains selection scoped so no detail polling occurs before explicit row selection. Pool-device slots poll only when enabled/mounted; the 3D server widget keeps its 10s slot override while its zpool list uses the restored 30s cadence.
