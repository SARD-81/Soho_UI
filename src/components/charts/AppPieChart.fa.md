# AppPieChart.tsx

## نمای کلی
کامپوننت `AppPieChart` یک پوشش برای کامپوننت PieChart از Material UI X است که برای ارائه استایل و پیکربندی پیش‌فرض برای نمودارهای دایره‌ای در برنامه طراحی شده است. این کامپوننت عملکرد پایه PieChart را با خصوصیات و گزینه‌های سفارشی استایل گسترش می‌دهد.

## ساختار فایل و اجزاء به تفصیل

### دستورات واردات
- `PieChart` و `PieChartProps` از '@mui/x-charts/PieChart': کامپوننت نمودار دایره‌ای پایه از Material UI
- `mergeChartSlotProps` از './chartStyles': تابع ابزارکاری برای ادغام خصوصیات شکاف نمودار با استایل‌های پیش‌فرض

### تعریف نوع (`AppPieChartProps`)
```ts
export type AppPieChartProps = PieChartProps & {
  disableDefaultLegend?: boolean;
};
```
- استاندارد PieChartProps را با یک گزینه اضافی گسترش می‌دهد
- `disableDefaultLegend`: بولین اختیاری برای کنترل نمایش قانونچه پیش‌فرض (به طور پیش‌فرض درست برای نمودارهای دایره‌ای است)

### تجزیه Props کامپوننت
```ts
const {
  slotProps,
  disableDefaultLegend = true,
  ...rest
}: AppPieChartProps
```
- props slotProps و disableDefaultLegend را استخراج می‌کند
- disableDefaultLegend را به طور پیش‌فرض به درست تنظیم می‌کند (متفاوت از سایر کامپوننت‌های نمودار)
- از rest برای ضبط تمام props دیگر برای ارسال به کامپوننت PieChart استفاده می‌کند

### ادغام خصوصیات شکاف
```ts
const mergedSlotProps = mergeChartSlotProps<PieChartProps['slotProps']>(slotProps, {
  includeLegend: !disableDefaultLegend,
}) as PieChartProps['slotProps'];
```
- props شکاف سفارشی را با استایل نمودار پیش‌فرض ادغام می‌کند
- تصمیم می‌گیرد که قانونچه را بر اساس ویژگی disableDefaultLegend وارد کند یا خیر
- از تأیید نوع برای حفظ تایپ صحیح استفاده می‌کند

### بازگشت JSX
```ts
return <PieChart {...rest} slotProps={mergedSlotProps} />;
```
- کامپوننت PieChart را با تمام props ارسال شده رندر می‌کند
- خصوصیات شکاف ادغام شده را که شامل استایل پیش‌فرض است اعمال می‌کند

## هدف و عملکرد
- کامپوننت Material UI X PieChart را با استایل پیش‌فرض پوشش می‌دهد
- یک گزینه برای غیرفعال کردن قانونچه پیش‌فرض فراهم می‌کند (به طور پیش‌فرض فعال=نادرست برای نمودارهای دایره‌ای)
- به طور خودکار استایل و تم ثابتی را در سراسر برنامه اعمال می‌کند
- خصوصیات سفارشی را با خصوصیات نمودار پیش‌فرض با استفاده از `mergeChartSlotProps` ادغام می‌کند
- API کامل کامپوننت اصلی PieChart را حفظ می‌کند در حالی که ویژگی‌های راحت را اضافه می‌کند

## Props
- `disableDefaultLegend`: بولین اختیاری برای کنترل دیده شدن قانونچه (به طور پیش‌فرض درست است، یعنی قانونچه به طور پیش‌فرض غیرفعال است)
- تمام props استاندارد `PieChartProps` به کامپوننت PieChart درونی منتقل می‌شوند

## ویژگی‌های کلیدی
- استایل و تم پیش‌فرض به طور خودکار اعمال می‌شود
- گزینه تغییر وضعیت دیده شدن قانونچه (رفتار پیش‌فرض متفاوت با سایر نمودارها)
- با سایر کامپوننت‌های نمودار در برنامه سازگار است
- سازگاری با تمام خصوصیات PieChart اصلی را حفظ می‌کند
- پشتیبانی RTL از طریق استایل‌های ادغام شده از chartStyles
- تایپ‌نویسی صحیح TypeScript

## الگوهای استفاده
- هنگامی که به یک نمودار دایره‌ای با استایل پیش‌فرض برنامه نیاز دارید استفاده کنید
- به طور پیش‌فرض، قانونچه غیرفعال است (نیازی به مشخص کردن disableDefaultLegend برای این رفتار نیست)
- هنگامی که می‌خواهید قانونچه را نشان دهید از `disableDefaultLegend={false}` استفاده کنید
- هر props استاندارد PieChart را برای داده، پیکربندی و غیره ارسال کنید
